# MySquire Frontend

Frontend do aplicativo **MySquire**, desenvolvido para gerenciamento completo de personagens de RPG.

A aplicação permite criar e organizar personagens, controlando atributos, equipamentos, magias, mapas e anotações em um único ambiente interativo.

## Funcionalidades

- Criação e edição de personagens
- Gerenciamento de status / atributos
- Sistema de equipamentos
- Controle de magias
- Visualização e interação com mapas
- Área de anotações do jogador
- Persistência de dados no cliente através de api 

## Backend
- Neon postgres
Link para api: https://github.com/Doki-glitch/MySquire

## Tecnologias

- React
- TypeScript
- Vite
- TailwindCSS
- React Router DOM
- React Hook Form
- Zustand (gerenciamento de estado)
- Konva / React-Konva (manipulação de mapas e canvas)
- Sonner (notificações)
- Lucide React / React Icons

## Bibliotecas e Dependências

O projeto utiliza diversas bibliotecas para melhorar a experiência e organização do código, incluindo:

- Gerenciamento de formulários com React Hook Form
- Estado global com Zustand
- Canvas interativo com Konva
- Estilização com TailwindCSS
- Ícones com Lucide React e React Icons

## Como executar

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/mysquire-frontend.git

cd mysquire-frontend

npm install

npm run dev


```
